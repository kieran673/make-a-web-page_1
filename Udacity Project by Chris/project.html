The web has four basic components:

HTTP: The main request-respond protocol of the web. It allows a client and a server to communicate. The client (for example a web browser) submits a HTTP request to a server (for example an application running on a computer hosting a web site) the server then provides resources to the client. Webpage: A hypertext document connected to the World Wide Web.

Web pages are written in HTML. HTML stands for Hypertext Markup Language. HTML documents form the majority of the content on the web. HTML documents contain text content which describes "what you see" and markup which describes "how it looks". This video gives a good overview.

Introduction to HTML

Elements

An element is made up a of start tag, the content and the end tag.They are written in the following format:

<tag> content </tag>

Tags

A tag is a code that describes how the contents must be formatted. Almost all tags have an opening and closing tag, EXCEPT void tags. A tag is created by putting the name of the tag in angled bracket, an end tag is the same but with a slash after the opening angled bracket and before the tag.
Tags can also be INLINE (the content is displayed without a new line break) or BLOCK (The content is displayed within its own box)
Below is a table giving details of just a few tags,

more like an "inline block")		 ✘

Attributes

Attrubributes are put inside the opening tag to modify them.
Here are just two examples:

<img src = "smily_face.jpg" alt = "a smily face" height = "42" width = "42">
  • src: Gives the source of the image. This tells the program where to get the picture
  • alt: If the picture is unable to load this tells the program what text to show instead.
  • height: Specifies the height of the image in pixels
  • width: Specifies the width of the image in pixels

<div class = "description example_class ThisIsAClass" id = "example">
  • class: gives the element a class which interests with several other things (such as a the CSS file) a div can have several classes seperated by a space.
  • id: Similar to class, however a element can only have ONE id.

There is an incredible ammount of tags and attributes. A good place to see them all is this site: http://www.w3schools.com/tags/default.asp

Structured documents

HTML is written in a tree like structure. What this means is that each HTML element can have a "parent" and a "child". When appearing in a browser these appear as nested boxes, we have boxes appearing in boxes appearing in boxes. This concept is especially important when dealing with styles as "children" will inherit their "parents" styles but any styles in the "child" with take priority.

How to Include CSS in a web page

There are three methods to include CSS in a web page

Method 1: Write your style inline with your HTML



Write the CSS inside the individual HTML element. This is not recommended because it requires a lot of repeated code but is useful if you want a certain section to have it's own style.

Method 2: Write CSS in the of the HTML



Inside the head element of the HTML add a style element. Write the CSS inside that style element.

Method 3: Link your HTML to a separate CSS file



Put all of your CSS code in a file separate from the HTML file. Add a tag inside the element of the TML file(Since is a "void tag" you don't need to add a closing) then add the following attributes to the : * rel="stylesheet" * type="text/css" *href="main.css"

Cascade

What happens if the HTML finds two styles rule that apply to the same section? This is where the order is important. Styles are applied in the following order with definitions further down the list overwriting previous definitions
  • the default style of a browser (different browsers have slightly different styles)
  • stylesheet in a separate file (this is what you will be mostly using)
  • stylesheet inside HTML (this can be done for small projects but is not ideal)
  • inline style in an element (this can also be done but should be avoided)

What is the difference betwen 2 + 2 and "2" +"2"?


In python 2 is a number and "2" is a string. This means python treats them differently. For example when using the + operator with numbers would add the numbers together but using them with strings would concatenate the strings. So 2 + 2 would give 4 whereas "2" + "2" would give "22".

Variables

A variable is a symbol representing a value that can be reused. The value can assigned with the "=" sign, for example My_variable = 2.
Variables give us a way to improve the readability of the code and reuse a value over and over again. We can also use them to record and follow changes. For example we can add 2 to My_variable with My_variable += 2. Now My_variable has the value of 4.

Lists

A data structure is a sequence of data. The most basic type of data structure is a list.
Lists are sequence of elements surrounded by square brackets for example ["one", "two", "three"]. Each element is assigned a number which is its position or "index". The first index is zero, the second is one, the third is two and so on.
This indexing allows us to call specific indexes from the list. In the example above we could use [1] to call "two" or [1:2] to call "two", "three".
What makes lists so useful is their mutability, The ability to change the list. A common method is append. Written as list.append(x) (where list is the list we are mutating and x is the value to be added) this allows us to add a new piece of data to a list.
Append is very similar to the operate +, however append will mutate a list but + will create a new list.

if/elif/else

We use if, elif and else to test if the conditions after the if and elif are true before executing the code. If the conditions are not met then that part of the code is skipped.
We use else if we want code that will execute if none of the conditions in if and elif are met. We use "if" first, followed by any number of "elif" and end with else. The order is important as once python finds a condition that is True it stops checking the other "if".

Loops

Loops are an extremely useful tool in python that repeat a section of code under certain conditions.there are two types of loops in python for and while

for:

These loops iterate over a given sequence.

while:

These loops repeat as long as a certain condition is met.

Function

A function is a piece of code, then when called (by using the name of the function) executes that code. A function must be made and then used. Functions are made by starting a line of code with the keyword def (which stands for define) and then giving a function name followed by the function parameters in parentheses. when used (or called) we write the name of the function but place actual values into the parentheses.
Functions are extremely useful for a programmer since they no longer need to keep rewriting a piece of code that they use over and over again.
A very important part of the function is the return statement, this is what produces the output, without this statement the function will run through the code but then will not actually give python anything to use.
